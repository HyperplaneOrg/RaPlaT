NAME

r.raplat - RaPlaT - raplat module (v30mar2023), RAdio PLanning Tool

KEYWORDS

SYNOPSIS

r.raplat
r.raplat --help
r.raplat [-rpcx] csv_file=string [antmap_file=string] dem_map=string [clutter_map=string] [region=string] [rx_ant_height=float] [generate=string] [bandwidth=float] [rx_threshold=float] out_map=string [cellnum=integer] [db_driver=string] [database=string] [out_table=string] [dbperf=integer] [procnum=integer] [freq_ovr=float] [radius_ovr=float] [model_ovr=string] [--overwrite] [--help] [--verbose] [--quiet] [--ui]

Flags:

-r
Reuse results from existing intermediate model/sector files
-p
(purge) Delete all unused sector radio coverage files
-c
(check) Test run without actually performing radio coverage computation
-x
Write all maps (model/sector/output) also as xyz files (in the working directory)
--overwrite
Allow output files to overwrite existing files
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

csv_file=string [required]
Radio cell/sector table in CSV format
antmap_file=string
Antennas map file
Default: $GISBASE/etc/radio_coverage/antenna_diagrams/antennamap.csv
dem_map=string [required]
DEM raster map for radio coverage simulation
clutter_map=string
Clutter raster map (required for HataDEM model)
region=string
Computation region (dem, current or region,raster,n,e,s,w,res (see g.region))
Default: current
rx_ant_height=float
Receiver antenna height [m]
Default: 1.5
generate=string
Selection of the generated output contents
Options: rss-max, coverage, rss-sum, rss-maxix, lte-rssi, lte-rsrp, lte-rsrq, lte-cinr, lte-maxspecteff, lte-maxthrput, lte-interfere
Default: rss-max
bandwidth=float
Bandwidth [MHz]
Default: 5
rx_threshold=float
Minimum received power [dBm] for radio signal coverage
out_map=string [required]
Simulated radio coverage - raster (output)
Default: out_raster
cellnum=integer
Number of succesive path loss values to be written in the table
Default: 5
db_driver=string
Database driver
Options: none, dbf, mysql, pg, sqlite, csv
Default: none
database=string
Database name
Default: $GISDBASE/$LOCATION_NAME/$MAPSET/dbf
out_table=string
Simulated radio coverage - db table (output)
Default: out_db
dbperf=integer
Database insert performance (rows/INSERT; 99: special fast mode via CSV)
Options: 1-99
Default: 20
procnum=integer
Number of parallel processes (-1: automatic, 0: non-parallel)
Default: -1
freq_ovr=float
Radio frequency override [MHz]
radius_ovr=float
Radius override [km]
model_ovr=string
Model override (with parameters)
