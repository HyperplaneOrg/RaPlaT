NAME

r.fspl - RaPlaT - Fspl module (v14aug2017)

KEYWORDS

raster, fspl

SYNOPSIS

r.fspl
r.fspl --help
r.fspl [-q] input_dem=name output=name [loss_exp=float] [loss_offset=float] coordinate=x,y [radius=float] [ant_height=float] [rx_ant_height=float] frequency=float [--overwrite] [--help] [--verbose] [--quiet] [--ui]

Flags:

-q
Quiet
--overwrite
Allow output files to overwrite existing files
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

input_dem=name [required]
Name of input raster map
output=name [required]
Name for output raster map
loss_exp=float
Exponent value (free-space-based fading model)
Default: 2.0
loss_offset=float
Offset value [dB] (free-space-based fading model)
Default: 0.0
coordinate=x,y [required]
Base station coordinates
radius=float
Computation radius [km]
Default: 10
ant_height=float
Transmitter antenna height [m]
Default: 10
rx_ant_height=float
Receiver antenna height [m]
Default: 1.5
frequency=float [required]
Frequency [MHz]
