NAME

r.sector - RaPlaT - Sector module (v06dec2018)

KEYWORDS

raster, directional diagram, antenna sector

SYNOPSIS

r.sector
r.sector --help
r.sector [-q] pathloss_raster=name input_dem=name output=name east=float north=float [radius=float] ant_data_file=string height_agl=float beam_direction=integer mech_tilt=integer [rx_ant_height=float] [--overwrite] [--help] [--verbose] [--quiet] [--ui]

Flags:

-q
Quiet
--overwrite
Allow output files to overwrite existing files
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

pathloss_raster=name [required]
Name of isotropic antenna pathloss raster map
input_dem=name [required]
Name of elevation model raster map - required for transmitter height determination
output=name [required]
Name for output raster map
east=float [required]
Easting coordinate
north=float [required]
Northing coordinate
radius=float
Computation radius [km]
Default: 10
ant_data_file=string [required]
Antenna data file
height_agl=float [required]
Above ground level height [m]
beam_direction=integer [required]
Beam direction [deg]
mech_tilt=integer [required]
Mechanical antenna tilt [deg]
rx_ant_height=float
Receiver antenna height [m]
Default: 1.5
